# 非機能要件

## 概要

本ドキュメントは、機能以外の要件（品質、運用、技術的な制約など）を定義する。
**「後からやる」は「やらない」と同義**。本ドキュメントで明確に決める。

## 詳細

### 認証・認可

| 項目 | 詳細 |
|------|------|
| 一般ユーザー | 認証不要（お気に入り等はlocalStorage保存） |
| 管理者認証 | **Basic認証**（/admin 配下） |
| 認証情報管理 | 環境変数で設定（具体的な変数名はPh5設計で決定） |

---

### SEO

| 項目 | 詳細 |
|------|------|
| meta情報 | 各ページに適切なtitle, descriptionを設定 |
| OGP | Twitter/Facebook等でのシェア時に画像・タイトル・説明を表示 |
| サイトマップ | 自動生成（フレームワークの機能を利用） |
| robots.txt | 適切に設定 |
| 構造化データ | カクテル詳細ページにRecipeスキーマ等を適用（余裕があれば） |

#### OGP対応詳細

| 項目 | 詳細 |
|------|------|
| og:title | カクテル名 / サイト名 |
| og:description | カクテルの説明 / サイト説明 |
| og:image | カクテル画像 / サイトロゴ |
| og:type | website / article |
| twitter:card | summary_large_image |

---

### アクセス解析

| 項目 | 詳細 |
|------|------|
| ツール | **Google Analytics 4 (GA4)** |
| 計測対象 | PV、ユーザー数、セッション等（デフォルト） |
| イベント | 必要に応じて追加（お気に入り登録等） |

---

### アクセシビリティ

| 項目 | 詳細 |
|------|------|
| 対応レベル | 最低限（CCにおまかせ） |
| キーボード操作 | 基本的なフォーカス移動は対応 |
| スクリーンリーダー | 画像にalt属性を設定する程度 |
| 色のコントラスト | 特に基準は設けない（CCの判断） |

---

### パフォーマンス

| 項目 | 詳細 |
|------|------|
| 目標値 | 特に設定しない（CCにおまかせ） |
| 画像最適化 | **対応する**（WebP変換、遅延読み込み等） |
| バンドルサイズ | 特に目標値なし |

---

### セキュリティ

| 項目 | 詳細 |
|------|------|
| 基本対策 | フレームワークのデフォルト対策に従う（XSS, CSRF等） |
| 追加対策 | 特になし |
| 脆弱性対応 | 依存パッケージは定期的に更新 |

---

### 多言語対応

| 項目 | 詳細 |
|------|------|
| 対応言語 | **日本語のみ** |
| 将来対応 | 今回は考慮しない |

---

### ログ・監視

| 項目 | 詳細 |
|------|------|
| エラーログ | ホスティングサービスのビルトイン機能を利用（Sentry等の導入はしない） |
| 死活監視 | **対応しない** |
| アラート通知 | なし |

---

### バックアップ

| 項目 | 詳細 |
|------|------|
| 対象 | データベース |
| 方式 | **ホスティングサービス任せ** |
| 頻度・保持期間 | ホスティングサービスのデフォルト設定 |
