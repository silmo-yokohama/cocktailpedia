## 制約条件

### 技術的制約

- 施主はフロントエンド経験あり（React等）
- バックエンド/インフラは非専門（丁寧な説明が必要）
- 無料枠を優先的に活用する

### 予算的制約

- 月額費用は無料枠内に収めることを目標とする
- 有料サービスは極力使用しない

### その他の制約

- 個人開発（チームメンバーなし）
- シンプルで保守しやすい構成を優先

## 用語集

| 用語 | 定義 |
|------|------|
| ベース（基酒） | カクテルの主成分となるお酒（ジン、ウォッカ、ラム、テキーラ、ウイスキー、ブランデー等） |
| 技法 | カクテルの作り方（シェイク、ステア、ビルド、ブレンド） |
| 閲覧回数 | カクテル詳細ページが閲覧された回数（サーバー側でカウント） |
| ブックマーク数 | カクテルがお気に入り登録された累計回数（サーバー側でカウント、匿名） |
| お気に入り | ユーザーがデバイスローカルに保存するブックマーク機能 |
| 閲覧履歴 | ユーザーが最近見たカクテルの履歴（デバイスローカルに保存） |
| 検索表示フラグ | 材料マスタにおいて、検索条件の「材料」に表示するかどうかのフラグ |
| 無限スクロール | ページを下にスクロールすると自動で次のコンテンツが読み込まれる方式 |
| 関連カクテル | ベースや材料の共通点から自動計算される類似カクテル |
