# 画面要件

## 概要

本ドキュメントは、Cocktailpedia の画面構成とデザイン方針を定義する。

## 画面詳細

### S-001: トップページ

| 項目 | 内容 |
|------|------|
| 概要 | カクテル一覧と最近見たカクテルを表示するメインページ |
| 表示内容 | ①ヘッダー、②最近見たカクテル（履歴がある場合）、③カクテル一覧、④フッター |
| ユーザーアクション | ソート変更、検索モーダル表示、カクテルカードクリック、お気に入り登録 |
| 対応機能 | F-001, F-003, F-007, F-013 |
| 遷移元 | 各画面 |
| 遷移先 | S-002, S-003, S-004, S-005, S-007 |
| 備考 | 無限スクロール（60件ずつ読み込み） |

---

### S-002: カクテル詳細

| 項目 | 内容 |
|------|------|
| 概要 | カクテルの詳細情報を表示 |
| 表示内容 | 画像、名前（日/英/別名）、ベース、材料、技法、グラス、度数、冷たさ、炭酸、カラー、由来、カクテル言葉、バリエーション、関連カクテル |
| ユーザーアクション | お気に入り登録/解除、関連カクテルクリック、戻る |
| 対応機能 | F-002, F-005, F-012 |
| 遷移元 | S-001, S-003, S-004, S-002（関連カクテル） |
| 遷移先 | S-002（関連カクテル） |
| 備考 | 閲覧時に閲覧回数カウント＋履歴に記録 |

---

### S-003: ベース別一覧

| 項目 | 内容 |
|------|------|
| 概要 | 指定されたベース（基酒）のカクテル一覧を表示 |
| 表示内容 | ベース名、カクテル一覧 |
| ユーザーアクション | ソート変更、検索モーダル表示（ベースは選択済み）、カクテルカードクリック |
| 対応機能 | F-004, F-003, F-007 |
| 遷移元 | ヘッダーナビ、トップページ |
| 遷移先 | S-002 |
| 備考 | URL例：`/filter/gin`, `/filter/vodka` 等 |

### S-007: 検索モーダル

| 項目 | 内容 |
|------|------|
| 概要 | 検索条件を入力するモーダル |
| 表示内容 | 検索条件フォーム（フリーワード、ベース、材料、技法、度数、グラス、冷たさ、炭酸、カラー） |
| ユーザーアクション | 条件入力、検索実行、条件クリア、モーダルを閉じる |
| 対応機能 | F-003 |
| 遷移元 | ヘッダーの検索アイコン |
| 遷移先 | 検索結果はS-001に反映 |
| 備考 | ベース別一覧（S-003）から開いた場合、ベースは選択済み状態 |

#### UI詳細

| 項目 | 表示形式 |
|------|---------|
| グラスの種類 | **アイコン選択**（各グラスをアイコンで表示、選択状態が分かるようにする） |

### S-101〜S-106: 管理画面

管理画面は簡易的なCRUD画面。公開画面ほどデザインにこだわる必要はないが、使いやすさを重視。

※ **Basic認証**でアクセス制限（専用ログイン画面は不要）

| 画面 | 主な機能 |
|------|---------|
| S-101 ダッシュボード | 統計情報表示（登録カクテル数、材料数、総閲覧回数など） |
| S-102 カクテル一覧 | 一覧表示、検索、編集/削除リンク、新規登録リンク |
| S-103 カクテル登録/編集 | フォーム入力、画像アップロード（※）、保存 |
| S-104 材料一覧 | 一覧表示、編集/削除リンク、新規登録リンク |
| S-105 材料登録/編集 | フォーム入力、保存 |
| S-106 サイト設定 | サイト説明文編集、保存 |

※ 画像アップロードの対応形式・サイズ上限はPh5設計で決定

## デザイン方針

### 全体の方向性

**クラシック・レトロ** - バーの雰囲気を感じさせる重厚感のあるデザイン

- 落ち着いた色調で大人の雰囲気を演出
- 適度な装飾でクラシカルな高級感を表現
- 余白を適切に使い、情報を見やすく整理
- カクテルの美しさを引き立てるデザイン

### カラー

| 項目 | 指定 |
|------|------|
| 基調色 | ダーク系（CCにおまかせ） |
| メインカラー | クラシック・レトロな雰囲気に合わせてCCにおまかせ（例：琥珀色、ゴールド、ディープレッド等） |
| アクセントカラー | CCにおまかせ |
| トーン | 重厚、落ち着き、高級感 |
